🔓 Unlock Em' EVOLVED (v3.0)

The ultimate "God-Mode" unlocker for Mindustry. This mod bridges the gap between campaign play and sandbox freedom, allowing you to use any block or unit on any planet at zero cost.

📜 The Legend of v3.0

This mod was originally a simple sandbox unlocker. During the development of the "Evolved" version, a terminal misclick (rm -rf) wiped the entire development environment. This version (v3.0) was rebuilt from scratch using chat logs and the combined brainpower of a three-way development team.

🚀 Features

Universal Unlocks: Every block in the game is unlocked and visible in the build menu.

Zero-Cost Building: All blocks have their requirements removed. Build anything, anywhere, for free.

Unit God-Mode: All units and factories are unlocked from the start.

Sandbox Integration: Brings Sandbox tools (Power Source, Item Source, etc.) into the campaign HUD.

Crash-Proof Logic: Optimized to run safely on Mindustry v146+ without Java field errors.

🛠️ Installation

Download the repository as a ZIP.

Extract the folder into your Mindustry mods/ directory.

Restart Mindustry and enable Unlock Em' EVOLVED.

Go to The Onset and start building your empire.
